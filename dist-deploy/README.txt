hand-marketing.ru — архив для развёртывания
=============================================

Файл: hand-marketing-site.zip  (≈290 МБ, 2674 файла)
Это полностью готовый статический сайт — автономная копия, без зависимости от Tilda.

КАК РАЗВЕРНУТЬ
-------------
1. Распаковать архив.
2. Загрузить ВСЁ СОДЕРЖИМОЕ архива в корень сайта на хостинге
   (обычно папка public_html или www).
   Важно: кладётся содержимое (index.html, static/, api/, .htaccess …),
   а не вложенная папка.
3. Открыть домен в браузере — сайт работает.

ФОРМЫ ЗАЯВОК
------------
Обработчик: api/lead.php (нужен PHP ≥ 7.4 на хостинге).
Перед запуском укажите получателя писем в api/lead.php — строка:
    $TO = 'info@hand-marketing.ru';
Заявки дублируются в api/leads.csv.

СЕРВЕР
------
- Apache: модули mod_rewrite, mod_headers, mod_expires (обычно включены).
  Настройки уже в .htaccess (красивые URL, кэш, gzip, заголовки).
- Nginx: отдавать статику, index.html как индекс каталога; для форм — php-fpm.

ПРОВЕРКА ПОСЛЕ ЗАГРУЗКИ
----------------------
[ ] Главная, кейсы, контакты открываются, графика на месте
[ ] Раздел «Проекты» показывает карточки (данные из api/*.json)
[ ] Тестовая заявка с формы — пришло письмо / появилась строка в api/leads.csv

Внешними остаются только: Яндекс.Метрика, Яндекс.Карты (на /contacts),
видео (Dropbox/YouTube). Обращений к Tilda — нет.
